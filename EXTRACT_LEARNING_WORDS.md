# 日语学习词汇提取器 (Learning Word Extractor)

## 简介
这是一个基于 SudachiDict 的 Python 脚本 (`extract_learning_words.py`)，旨在帮助日语学习者从文章中自动筛选出**难易适中、具有学习价值**的单词。

它能够自动过滤掉：
- 过于简单的高频词（如“吃”、“去”、“我”）。
- 过于生僻或无需背诵的专有名词（如人名、地名、机构名）。
- 无实际意义的助词、助动词等。

从而精准定位出**中级进阶词汇**（如“感染症”、“功能性”等）。

## 核心原理

该工具利用 SudachiDict 不同版本词典的收录范围差异来判断词汇的“基础程度”：

| 词汇类型 | 判断依据 | 示例 | 学习建议 |
| :--- | :--- | :--- | :--- |
| **基础词 (Basic)** | 存在于 `Small` 词典中 | 私, 食べる, 昨日 | 高频基础词，通常无需刻意记忆 |
| **进阶词 (Advanced)** | **不在 `Small` 但在 `Full` 中的普通名词/动词** | **感染症, 機能性** | **⭐️ 重点学习 (核心中频词)** |
| **专有名词 (Proper)** | 词性为 `固有名詞` | 鬼滅の刃, 東京都 | 仅需识别，无需死记硬背 |
| **忽略 (Ignore)** | 助词/助动词/数词等 | の, た, 一つ | 语法功能词，不作为单词记忆 |

## 环境准备

你需要安装 SudachiPy 以及 `small` 和 `full` 两个版本的词典：

```bash
pip install sudachipy sudachidict_small sudachidict_full
```

## 使用方法

### 1. 命令行运行

直接运行脚本即可看到内置示例的分析结果：

```bash
python3 extract_learning_words.py
```

### 2. 在代码中调用

你可以将 `LearningWordExtractor` 类导入到你自己的项目中：

```python
from extract_learning_words import LearningWordExtractor

extractor = LearningWordExtractor()
text = "私は昨日、東京で鬼滅の刃の映画を見ました。"
results = extractor.extract(text)

# results 是一个包含字典的列表
for word in results:
    if word['level'] == '进阶':
        print(f"推荐背诵: {word['surface']} ({word['reading']})")
```


## 字段说明

- **词面 (Surface)**: 单词在原文中的写法。
- **读音 (Reading)**: 单词的片假名读音，可用于生成振假名。
- **原形 (Base)**: 动词/形容词的辞书形，方便查词典。
- **词性 (POS)**: 详细的词性分类。
- **级别 (Level)**: `基础`, `进阶`, `专名`。
- **建议 (Suggestion)**: 针对该词的学习建议。
